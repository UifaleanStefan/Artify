Verificare 100% – checklist

Bifează fiecare pas. Dacă toate sunt OK, pixelul funcționează 100%.

  1. Config: mai sus apare „Pixel ID din config: ••••xxxx” (verde). Dacă e roșu, setează FACEBOOK_PIXEL_ID în .env și repornește serverul.
  2. Pixel încărcat: apare „Pixel încărcat: da” (verde).
  3. Script în pagină: F12 → Network → filtrează „facebook” → reîncarcă → trebuie request la connect.facebook.net.
  4. Meta Pixel Helper: instalează, reîncarcă pagina, apasă pe iconița extensiei → vezi PageView și ViewContent (verde). După click pe buton, apare încă un ViewContent.
  5. Events Manager: Events Manager → Website → Test Events. Deschide acest URL în browser; în 20–60 sec ar trebui să apară evenimente. Asta confirmă că Facebook primește datele.

Concluzie: Dacă 1–2 sunt verzi și 4 sau 5 confirmă în Helper / Events Manager, pixelul funcționează 100%.

Pe homepage (landing) apare „No Pixels found”?

  • View Source pe pagina principală (/) și caută „Facebook Pixel Code”. Dacă nu apare, răspunsul e din cache sau e servit static (CDN/host) și pixelul nu e injectat.
  • Incognito + fără extensii: unele extensii (ad blocker, privacy) blochează connect.facebook.net. Testează în fereastră incognito cu extensiile dezactivate.
  • Header: la request la /, răspunsul ar trebui să aibă header X-Pixel-Injected: 1 (vezi în F12 → Network → click pe document /).
