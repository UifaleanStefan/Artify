Eveniment trimis. Verifică în Meta Pixel Helper sau în Events Manager → Test Events.

Cum verifici

  • Meta Pixel Helper (Chrome): instalează extensia, deschide această pagină și apasă pe iconița extensiei. Ar trebui să vezi PageView și (după click) ViewContent.
  • Events Manager: sursa Website → Test Events. Deschide acest URL în browser; evenimentele apar în ~20 sec.
  • Network: F12 → Network → filtrează „facebook” sau „fbevents”; la încărcare ar trebui să apară request către connect.facebook.net.
